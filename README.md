# Egzamin Zawodowy - INF.04

Egzamin zawodowy INF.04, czyli _Projektowanie, programowanie i testowanie aplikacji_, to drugi, kluczowy egzamin potwierdzający kwalifikacje w zawodzie Technik Programista. Skupia się on na tworzeniu bardziej złożonego oprogramowania, wykraczającego poza obszar stron internetowych, obejmując technologie desktopowe oraz mobilne, a także profesjonalne procesy testowania i dokumentowania kodu.

Zakres tematyczny INF.04 obejmuje między innymi:

- Projektowanie i programowanie aplikacji desktopowych (np. przy użyciu języka C# z wykorzystaniem platformy .NET, Java czy Python).
- Tworzenie aplikacji mobilnych (np. na system Android).
- Projektowanie algorytmów i wykorzystywanie zaawansowanych struktur danych.
- Testowanie oprogramowania, debugowanie oraz zapewnianie jakości kodu.
- Tworzenie pełnej dokumentacji technicznej aplikacji oraz instrukcji dla użytkownika.
